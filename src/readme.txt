Программирование микроконтроллера через ардуинку.
HEX-файл необходимо преобразовать в BIN-файл (hex2bin.exe, найдена на просторах интернета), а затем в массив для скетча (т.к. в оперативную память заливаемая программа не влазит и поэтому будем её рамещать во флэш-памяти). Автоматизации подготовлен коммендный файл (_make_h_from_hex.bat).
Для преобразования из бинарного файла в массив для "с" написана маленька утилитка (bin2h.exe, откомпирована mingw). Исходники приложены.
Можно обойтись и без релюшки, достаточно руками имитировать ее работу.
